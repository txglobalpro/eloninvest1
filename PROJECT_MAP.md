# PROJECT_MAP - ElonInvest Platform

## [TECH_STACK]
- **Runtime**: Python 3.12.2
- **Framework**: Flask 3.1.3
- **Database**: SQLite (Flask-SQLAlchemy 3.1.1)
- **Auth**: Flask-Login 0.6.3
- **Email**: Flask-Mail 0.10.0
- **Migrations**: Flask-Migrate 4.1.0
- **Frontend**: Bootstrap 5.3.x (CDN), Bootstrap Icons 1.11.x (CDN), Chart.js 4.x (CDN)
- **Scheduler**: APScheduler (daily profits)
- **i18n**: Custom JSON translations (AR/EN)
- **Forms**: WTForms
- **Password**: Werkzeug 3.1.8 (generate_password_hash/check_password_hash)
- **Theme**: CSS Variables (dark/light) + localStorage persistence

## [SYSTEM_FLOW]
```
Visitor → Register → Email Verification → Login → Profile (update settings)
         → Homepage: Hero + Features + Licenses (FCA/CySEC/FinCEN/DFSA)
                     + Random Testimonials (avatar + quote, rotates 5s)
                     + Floating Toast Notifications (bottom-left, 8-12s)
         → Select Plan (with live calculator) → Deposit (PM cards + copy address + network badge)
         → Daily Profit (APScheduler)
         → Withdraw (wallet overview + form + history + info cards)
         → Referral Link → Level 1 (5%) → Level 2 (3%) → Level 3 (2%)
         → Rewards: Welcome | First Deposit | Loyalty | Daily Login Streak
         → Market: Stock prices (random walk) + TSLA Chart
         → Admin Panel: Stats | Users (search + detail) | Plans (CRUD) | Payment Methods (CRUD) | Transactions (approve/reject deposits & withdrawals)
```
Deposit Workflow: user submits pending → admin approves → balance credited + referral commissions + first-deposit reward
Withdraw Workflow: user selects PM card → picks network → enters address → enters amount → submits pending (no balance deducted) → admin approves (deducts balance, sees method/network/address in note) or rejects (no refund)

## [DATABASE MODELS]
- **User**: id, username, email, password_hash, balance, profit, deposits, withdrawn, email_verified, **verification_token**, is_admin, **is_banned**, referral_code, referred_by, first_deposit_date, daily_streak, last_daily_claim, lang, payment_method (per-user saved info), created_at, last_login
- **Plan**: id, name, company, min_amount, max_amount, daily_roi, duration_days, description
- **PaymentMethod**: id, type (crypto/paypal/bank/card/other), label, details, is_active, created_at
- **Investment**: id, user_id, plan_id, amount, start_date, end_date, status
- **Transaction**: id, user_id, type, amount, status, note, withdraw_method, withdraw_network, withdraw_address, created_at
- **Referral**: id, referrer_id, referred_id, level, commission, created_at
- **Reward**: id, user_id, type, amount, claimed_at

## [ADMIN PANEL]
| Page | Route | Features |
|------|-------|----------|
| Dashboard | `/admin/` | Stats overview, quick links |
| Users | `/admin/users` | List all users (JS search bar) with email, balance, plan, payment → link to detail → Ban/Delete buttons |
| User Detail | `/admin/user/<id>` | Full mgmt: balance adjust/set, password, admin toggle, verify email, payment method, change plan, view transactions |
| Plans | `/admin/plans` | CRUD: add/edit/delete plans with all fields (name, company, min/max, ROI%, duration, description) |

| Transactions | `/admin/transactions` | List all (deposits + withdrawals + pending), Email column, Deposited Before indicator (green check / red X), approve deposits (credits balance + referral + first-deposit reward), approve withdrawals (deducts balance), reject (no refund) |
